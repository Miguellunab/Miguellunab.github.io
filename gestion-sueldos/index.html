Gestión de Sueldos

Módulo web para administrar usuarios, roles y cálculos salariales. Accede al panel seguro desplegado en la nube para realizar operaciones de manera eficiente y confiable.

Flask PostgreSQL Docker AWS Autenticación JWT

Características Principales

El sistema está diseñado para escalar y mantener la integridad de los datos clave de nómina.

  • Gestión de empleados y roles
  • Cálculo automático de horas y bonificaciones
  • Historial y auditoría de movimientos
  • Reportes descargables (PDF / CSV)
  • Panel responsivo y seguro

Nota sobre la Integración

El servicio corre en un servidor externo. Para seguridad y evitar problemas de CORS y credenciales, se accede en una pestaña nueva. Si en el futuro migras a HTTPS podrás incorporar un iframe seguro o un reverse proxy.
