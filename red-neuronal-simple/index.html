Red Neuronal Simple

Conversión de Celsius a Fahrenheit usando TensorFlow.js

Conversor de Temperatura

Esta aplicación utiliza una red neuronal entrenada para convertir grados Celsius a Fahrenheit. Mueve el deslizador para probar diferentes valores:

0 grados Celsius son 32 grados Fahrenheit

¿Cómo funciona?

Esta red neuronal ha sido entrenada para aprender la relación entre grados Celsius y Fahrenheit. La fórmula matemática real es:

F = C × 1.8 + 32

Sin embargo, la red neuronal aprende esta relación por sí misma a partir de ejemplos, sin ser programada explícitamente con la fórmula. Durante el entrenamiento, la red ajusta sus pesos internos para minimizar el error de predicción.

Tecnologías utilizadas

  • TensorFlow.js para inferencia en el navegador
  • Python con TensorFlow/Keras para el entrenamiento inicial
  • HTML/CSS/JavaScript para la interfaz web

Estructura de la red

Este modelo utiliza una red neuronal sencilla con:

  • Una capa de entrada (1 neurona para el valor en Celsius)
  • Una capa oculta densa (4 neuronas con activación ReLU)
  • Una capa de salida (1 neurona para el valor en Fahrenheit)
